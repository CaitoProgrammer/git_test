Esto es un HEADING.

Sí, se define a HEADING cómo un subtitulo el cual denota la importancia que nosotros le damos a respectivos párrafos. Por ejemplo el título "¡Hola Mundo!" es un heading de nivel 1, es el heading de mayor importancia.

¿Cómo crear párrafos?

Para poder crear un párrafo al momento de agregar un elemento en tu código, es necesario poner el respectivo comando.

En este caso, tal comando se escribe de la siguiente manera: "< p >"

Destacar un conjunto de palabras dentro de un párrafo.

Para aplicar la propiedad de "negrita" a cierto conjunto de palabras debemos utilizar el comando "STRONG"el cuál como verán, permite destacar una palabra o varias. También podemos utilizar el comando "EM"para aplicarle énfasis a una o varias palabras.

¿Cómo crear una lista?

Utilizamos el comando "UL" determinando cada item con el comando "LI" y obtenemos:

  • Lista desordenada 1
  • Lista desordenada 2
  • Lista desordenada 3
Utilizamos el comando "OL"determinando cada item con el comando ""LI y obtenemos:
